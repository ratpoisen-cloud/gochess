# Go Chess logo assets

Use these exact file names for the current brand assets:

- `gochess_wordmark_light.svg` — main SVG for dark themes/backgrounds
- `gochess_wordmark_dark.svg` — main SVG for light themes/backgrounds
- `gochess_wordmark_light_1600.png` — PNG fallback for dark themes/backgrounds
- `gochess_wordmark_dark_1600.png` — PNG fallback for light themes/backgrounds

Optional higher-resolution PNG sources are also available:
- `gochess_wordmark_light_3200.png`
- `gochess_wordmark_dark_3200.png`
